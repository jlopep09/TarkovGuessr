# Tarkov Guessr

## Run the app
We can easily run the whole with only a single command:
```bash
docker compose up
```
## ui folder .env
You must create a .env file with the following variables

```bash
MONGODB_USER=
MONGODB_PASSWORD=
MONGODB_DATABASE=
MONGODB_LOCAL_PORT=
MONGODB_DOCKER_PORT=

SPRING_LOCAL_PORT=
SPRING_DOCKER_PORT=
```
